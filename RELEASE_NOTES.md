# 智净大师 Release Notes

## v1.0.0 (2026-05-16)

🎉 首个公开版本。

### 功能

- 任意路径扫描（支持多路径分号分隔）
- 智能文件分类（临时文件、日志、构建产物、备份、下载残留等）
- 空文件夹独立检测与合并建议
- 文件定位（一键打开所在文件夹并高亮文件）
- 多种清理模式（回收站、永久删除、安全擦除）
- 路径收藏与历史记录
- 扫描进度实时展示
- 深色模式支持

### 技术

- Tauri v2 + React 19 + Rust 后端
- 高性能并行文件遍历（jwalk）
- SQLite 本地持久化
- 跨平台架构（Windows/macOS/Linux）
